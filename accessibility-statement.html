Accessibility Statement

Our Commitment

Cloud Rescue Foundation is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

We believe that everyone, regardless of ability, should be able to access information about supporting rescue dogs and making a positive impact in animal welfare.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA.

This website is fully conformant with WCAG 2.1 Level AA standards.

"Fully conformant" means that the content fully conforms to the accessibility standard without any exceptions.

Accessibility Features

Our website includes the following accessibility features:

  • Keyboard Navigation: All functionality is available using a keyboard
  • Screen Reader Support: Compatible with NVDA, JAWS, VoiceOver, and TalkBack
  • Clear Focus Indicators: Visible focus outlines on all interactive elements
  • Skip Links: Quick navigation to main content
  • Color Contrast: All text meets WCAG AA contrast ratios (minimum 4.5:1)
  • Resizable Text: Text can be resized up to 200% without loss of functionality
  • Alternative Text: Descriptive alt text for all meaningful images
  • ARIA Labels: Proper labels and roles for assistive technologies
  • Language Support: Bilingual content (English and Chinese) with proper language tags
  • Reduced Motion: Respects user's motion preferences
  • Touch Targets: All interactive elements are at least 44x44 pixels on mobile
  • Responsive Design: Works on all devices and screen sizes

Assistive Technologies

This website is designed to be compatible with the following assistive technologies:

  • Screen readers (NVDA, JAWS, VoiceOver, TalkBack)
  • Screen magnification software
  • Voice recognition software
  • Keyboard-only navigation
  • Alternative input devices

Browsers and Devices

Our website has been tested with the following combinations:

  • Chrome, Firefox, Safari, and Edge (latest versions)
  • Windows 10/11 with NVDA and JAWS
  • macOS with VoiceOver
  • iOS with VoiceOver
  • Android with TalkBack

Technical Specifications

Accessibility of this website relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML5
  • WAI-ARIA (Web Accessibility Initiative - Accessible Rich Internet Applications)
  • CSS3
  • JavaScript

These technologies are relied upon for conformance with the accessibility standards used.

Assessment Approach

Cloud Rescue Foundation assessed the accessibility of this website by the following approaches:

  • Self-evaluation: Internal accessibility audits
  • Automated testing: WAVE, axe DevTools, Lighthouse
  • Manual testing: Keyboard navigation and screen reader testing
  • Code review: WCAG 2.1 Level AA compliance verification

Continuous Improvement

We are committed to maintaining and improving the accessibility of our website. We regularly review our website and make updates to ensure ongoing compliance with accessibility standards. This includes:

  • Monthly accessibility audits
  • User testing with people with disabilities
  • Staff training on accessibility best practices
  • Monitoring updates to WCAG guidelines
  • Promptly addressing reported accessibility issues

Legal Information

This website complies with:

  • Americans with Disabilities Act (ADA) - Title III
  • Section 508 of the Rehabilitation Act
  • WCAG 2.1 Level AA (internationally recognized standard)

Standard: WCAG 2.1 Level AA
